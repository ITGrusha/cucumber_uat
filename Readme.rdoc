= Quick Start 
== Requirements

I sucessfully managed to get it running on my system using the following setup

* Mac OS X 10.6
* Rubygems version >= 1.3.7
* bundler gem
* Firefox

It should also work on Ubuntu.

== Installation
sudo gem install bundler

sudo bundle install

== Usage

Run "cucumber CUCUMBER_HOST=http://www.spiegel.de/" on the command line
